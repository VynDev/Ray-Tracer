Implementation from the book: https://raytracing.github.io/books/RayTracingInOneWeekend.html

## Build & Run

### Using g++

```
g++ source/main.cpp -o RayTracer
```

### Using cmake & g++
```
mkdir build && cd build && cmake .. && make && cd -
```

Then just launch RayTracer
